# shopdb — Base de données E-commerce (SQL)

Base de données relationnelle complète pour une plateforme e-commerce.

## Contenu
- Gestion des utilisateurs et rôles
- Catalogue produits & catégories
- Panier, commandes, paiements
- Facturation et livraisons
- Gestion de stock
- Reporting via vues SQL
- Procédures stockées et triggers métier

## Technologies
- SQL (MySQL / MariaDB)
- InnoDB
